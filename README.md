## 🚀 Descrição do Projeto

Este projeto consiste em uma aplicação web de currículo desenvolvida utilizando **Blazor WebAssembly**. A aplicação exibe informações pessoais e experiências profissionais de forma dinâmica, utilizando componentes reutilizáveis.

O sistema apresenta:

* Perfil do usuário com imagem e descrição
* Lista de experiências profissionais renderizadas dinamicamente
* Organização em componentes para melhor manutenção

---

## 🛠️ Tecnologias Utilizadas

* **.NET**
* **Blazor WebAssembly**
* **C#**
* **HTML / CSS / Bootstrap**

## ▶️ Guia de Execução

Siga os passos abaixo para rodar o projeto via terminal:

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/seu-repositorio.git

# Acesse a pasta do projeto
cd seu-repositorio

# Restaure as dependências
dotnet restore

# Execute o projeto
dotnet run
```

Após isso, abra o navegador no endereço exibido no terminal (geralmente: `https://localhost:5001` ou similar).
